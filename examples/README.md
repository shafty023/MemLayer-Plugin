# MemLayer Examples

Working examples and configuration templates for MemLayer.

## Contents

### [Config Templates](configs/)

Ready-to-use MCP configuration files for every supported client:

- Claude Code
- Cursor
- Windsurf
- Cline
- VS Code Agent Mode

### [Quickstart](quickstart/)

Minimal example demonstrating the complete MemLayer learning loop:

- Retrieve context before starting work
- Log episodes after completing work
- Store permanent knowledge as notes
- Full workflow with pattern detection and reflection

## Getting Started

1. Copy the [config template](configs/) for your AI coding tool
2. Follow the [quickstart](quickstart/) to run your first learning loop
3. Read the [full documentation](https://prociq.ai/docs/introduction) for deeper coverage
