# Python MCP Client + MemLayer

Connect to MemLayer directly from Python using the MCP SDK.

## Prerequisites

```bash
pip install mcp
```

## What This Example Does

1. Opens a streamable HTTP connection to the MemLayer MCP endpoint
2. Retrieves context based on a task description
3. Simulates work
4. Logs the episode outcome

## Run

```bash
python memlayer_client.py
```

On first run, OAuth will open a browser window for sign-in.
